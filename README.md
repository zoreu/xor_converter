# xor_converter
xor file converter

proteja arquivos de videos contra remoção criptografando com chave

baixe o zip: https://github.com/zoreu/xor_converter/raw/refs/heads/main/xor_converter.zip

extraia o zip em uma pasta onde vai usar

-----------------------------------------------

requisitos: 

python 3.11+

vlc instalado

windows 10 ou windows 11

-----------------------------------------------

rode o arquivo CRIPTOGRAFAR_VIDEO.bat e defina o nome do arquivo e chave de criptografia

hospede em algum lugar pode ser archive mesmo

para rodar videos use o STREAM XOR.bat

coloque a url e a chave e seja feliz.
